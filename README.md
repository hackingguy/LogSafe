## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Sarthak480/LogSafe.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

## Usage

```npm run dev```
